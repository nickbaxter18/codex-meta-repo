# U-DIG-IT Hub Documentation

This directory is reserved for project-specific notes, diagrams, and runbooks for the equipment rental platform. Suggested structure:

- `architecture.md` - System diagram, key services, data flow.
- `agents.md` - Prompts, tool definitions, and guardrails for rental agents.
- `operations.md` - Deployment steps, smoke tests, on-call procedures.

Feel free to break files down further (e.g., `voice.md`, `realtime.md`) as the platform grows. Keeping this folder updated ensures Cursor and collaborators always have current context.
